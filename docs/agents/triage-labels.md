# Triage Labels

Matt Pocock 默认 5 类：

| 标签 | 含义 |
|---|---|
| `status: ready-for-agent` | 需求清晰 |
| `status: needs-triage` | 信息不全 |
| `status: in-progress` | agent 已开 PR |
| `status: blocked` | 依赖外部 |
| `status: needs-human-review` | 需 owner 验收 |
